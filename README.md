## Methodology

Each stock is evaluated on:

- **Asymmetric upside** — is there a catalyst within 12-18 months that changes the financial math?
- **Analyst consensus** — ratings, price targets, and conviction from Wall Street research
- **Fundamental quality** — revenue growth, operating margins, free cash flow, balance sheet strength
- **Technical context** — price relative to moving averages, entry zones, take-profit levels
- **Counter-thesis** — what the bears get right, not just what we want to be true

Sources include Yahoo Finance API (live prices, fundamentals, analyst data), industry reports, and independent research.
